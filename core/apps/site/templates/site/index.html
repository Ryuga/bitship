Main Features

User Friendly CMS

All your project files and variables can be managed from the CMS, No need for git.

3 Step Deployment

You can deploy your apps to heroku with just 3 steps using GUI, No git knowledge required.

Seamless Scaling & Cloning

Apps can be scaled, cloned or redeployed seamlessly with just the click of a button

Free as in Freedom
